┌─────────────────────────────────────────────────────────────────────────────────┐
│ المستخدمة العادية (Customer) │
└─────────────────────────────────────────────────────────────────────────────────┘

1. فتح التطبيق → الصفحة الرئيسية (index.html)
2. البحث:
- تكتب في شريط البحث (ولاية، بلدية، تخصص، اسم)
- تستخدم الفلاتر (السعر، التقييم، الأقرب، "متاحة الآن")
- تبديل بين عرض "قائمة" و "خريطة"
3. تختار حساباً تجارياً → تشوف صفحة الحساب (business-profile.html):
- تشوف BIO، الخدمات والأسعار، مواعيد العمل، العنوان (مع Google Maps)
- تشوف ألبوم الصور (قبل/بعد، أعمال، منتجات)
- تشوف فيديو 15 ثانية
- تشوف التقييمات (نجوم ذهبية/خضراء/حمراء)
4. تتابع الحساب (Follow) → تظهر منشوراته في الصفحة الرئيسية
5. تضيف الحساب إلى المفضلة (Favorite) لحفظه
6. تحجز خدمة:
- تختار نوع الخدمة (موعد صالون، فستان، منتج، خدمة منزلية)
- تحدد التاريخ والوقت
- ترسل طلب الحجز
7. بعد الموعد بساعتين، تصلها إشعار تطلب تقييمها → تملأ التقييم (4 تقييمات فرعية + تعليق + صور اختيارية)
8. إذا أرادت الإبلاغ → تذهب إلى صفحة الحساب → زر "إبلاغ"

┌─────────────────────────────────────────────────────────────────────────────────┐
│ صاحبة الحساب التجاري (Business Owner) │
└─────────────────────────────────────────────────────────────────────────────────┘

1. تفتح التطبيق → تضغط "إنشاء حساب تجاري" (business-register.html)
2. تملأ البيانات:
- نوع الحساب (صالون، حلاقة منزلية، متنقلة، بائعة، مؤجرة، خدمات منزلية)
- اسم الحساب، اسم المستخدم، BIO (≤100 كلمة)
- ترفع صور: صورة الحساب، صور وصفية، ألبوم (قبل/بعد، أعمال، منتجات)
- ترفع فيديو 15 ثانية
- تحدد العنوان: بلدية ← دائرة ← ولاية + وصف + تحديد Google Maps
- للحلاقة المتنقلة: تحدد مدى التنقل بالكيلومتر
- توثق الهوية: صورة الوجه + صورة ظهر البطاقة
- (اختياري للصالونات) ترفع السجل التجاري للحصول على "علامة موثوق"
3. تنتظر قبول الأدمن → تصلها إشعار بالقبول
4. لوحة التحكم (business-dashboard.html):
- تشوف الحجوزات الواردة (تقبل، ترفض، تكمل)
- تنشر منشورات (نص ≤500 كلمة + صور)
- ترفع قصص (ذهبي/بلاتيني فقط)
- تشوف التقييمات وترد عليها (رد واحد لكل تقييم)
- تشوف الإحصائيات (ذهبي/بلاتيني)
5. تختار الاشتراك:
- فضي (0 دج): 3 حجوزات يومياً، بدون تخفيضات
- تطلب ترقية إلى ذهبي (3000 دج/3 شهور) أو بلاتيني (6000 دج/6 شهور):
- ترفع صورة وصل التحويل إلى CCP
- الأدمن يقبل → يصبح ذهبي/بلاتيني
6. إذا تلقت 3 بلاغات → تحذير آلي → إذا كانت ذهبي/بلاتيني تطلب مراجعة الحذف

┌─────────────────────────────────────────────────────────────────────────────────┐
│ الأدمن (Admin) │
└─────────────────────────────────────────────────────────────────────────────────┘

1. يدخل لوحة الأدمن (admin-dashboard.html) عبر رابط خاص (يُمنح يدوياً)
2. يرى قائمة "حسابات قيد المراجعة":
- يعاين صور الهوية (الوجه والظهر)
- يقبل → يصبح الحساب مفعلاً
- يرفض → يكتب سبب الرفض
- بعد القبول، تُحذف صور الهوية من الخادم (server-side) ويُسجل وقت الحذف
3. يرى قائمة "حسابات نشطة":
- حظر / رفع حظر / حذف نهائي
4. يرى طلبات مراجعة التحذيرات (ذهبي/بلاتيني فقط):
- يقبل إزالة التحذير أو يرفض
5. يرى طلبات الدفع:
- يفتح صورة وصل التحويل
- يقبل → يحدّث subscription_tier و subscription_expires_at
- يرفض → يكتب سبب الرفض
6. يرى الإحصائيات: عدد المستخدمات العاديات، الحسابات التجارية، التقييمات، الإبلاغات
7. يتلقى إشعارات عبر Gmail عند:
- تسجيل حساب تجاري جديد في انتظار المراجعة
- طلب دفع جديد
- إبلاغ جديد (بعد 3 بلاغات يظهر تحذير آلي)
